Audacity

Audacity allows editing sound. See Music.html for creating music.

Install Audacity

For Windows:
https://www.audacityteam.org/download

For Ubuntu:
Install Audacity using Synaptic or command line.


  sudo apt-get install audacity

Open a Sound File

For Ubuntu:
  • Start Audacity using Activities and enter Audacity in the "Type to search box".
  • If Audacity is not available, install Audacity
Use one of the following to open a sound file (.mp3, .wav, .ogg, etc.).
  • Select File/Open, then search for the image.
  • If a folder is showing the image filename, press the mouse button down on the image, move the mouse to Audacity, and then release the mouse button.

Get Part of a Sound File

  • The main part of the Audacity program displays a picture of the sound.
  • Press the mouse button down on the picture, move the mouse, and then release the mouse button.
  • Press the Play button (green triangle) to hear the selection. A new area can be selected to get a different part of the sound file.
  • Select the part to delete before the part to keep and select the Edit/Cut or Edit/Delete menu options.
  • Select the part to delete after the part to keep and select the Edit/Cut or Edit/Delete.

Generate a Sound

Audacity can generate sounds using the Generate menu. There are Generate/Tone, Generate/Chirp, etc.
