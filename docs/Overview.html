At the moment, this documentation mostly describes using Ubuntu. Some changes may be required to run on Windows or other operating systems.

Use JupyterLab to run on other systems. Open a terminal in the codeplayfield directory. 

  cd codeplayfield
  ~/.local/bin/jupyter-notebook

Demos

Pictures

The pictures folder requires installing the "opencv" package.

Draw an Image


Open a terminal move to the "fields/pictures" folder and enter: 

    $ python3 1-drawImage.py
