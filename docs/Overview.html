CodePlayField

CodePlayField provides a way to learn audiovisual editing and programming using Python 3 using many simple demonstration programs.

The following video was created from the software. The video can be played in some browsers by right clicking on the video. The demo that created this allows using the arrow keys to move people around depending on where they are in the scenes.



The software can be run in few different ways.

  • Use Jupyter Notebooks. Installing packages is simpler since it is done within the notebooks.
  • Use a simple editor and run Python using a terminal or from Idle. This documentation mostly describes using Ubuntu. Some small changes may be required to run on Windows or other operating systems.

Install Jupyter
The recommended method to install Jupyter is to first install pip.
Install Jupyter using https://jupyter.org/install and use the pip method.


Run Jupyter

  cd codeplayfield
  jupyter-notebook CodePlayField.ipynb

In Ubuntu, jupyter notebooks is normally installed at "~/.local/bin/". 

  cd codeplayfield
  ~/.local/bin/jupyter-notebook CodePlayField.ipynb
