Rosegarden

Rosegarden allows editing music notation.

Install Rosegarden

For Windows:
https://sourceforge.net/projects/rosegarden/

For Ubuntu:
Install Rosegarden using Synaptic or command line.


  sudo apt-get install rosegarden

Rosegarden with sound

See Music.html for running Rosegarden with sound.

Edit Notes with Rosegarden

Some Rosegarden help is here:
https://www.rosegardenmusic.com/wiki/doc:notation-en
http://rosegarden.sourceforge.net/tutorial/en/chapter-8.html
  • Double click on a track to display the notation editor.
  • Use pencil to enter notes. The icon tools at left allow changing time and flat/sharp.
  • Use pointer to select many notes.
  • Add slur by selecting multiple notes, then Phrase/Add Slur
  • If you add a note to wrong place, it can be dragged to other place

Add a measure

In the main screen where the tracks are displayed:
  • Use click and drag to copy a track.
  • Click at the end of a track and drag to add measures. The cursor changes to a double arrow cursor if the cursor is at the right part of where the track ends.

Note dynamics (volume)

Rosegarden and MIDI call this "note velocity". There is a tool icon called "Adjust velocity to follow text dynamics". https://www.rosegardenmusic.com/doc/en/nv-text.html

Click on the "T" button, then click on the note where the dynamics should be placed below.

Save Rosegarden to Sound File

  • Start Audacity
  • Select menu Edit Preferences
  • Select MIDI Devices
  • Select Host: ALSA
  • Start recording (Red Circle)
  • Press Play in Rosegarden
